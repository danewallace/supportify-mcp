Making Apple Support guides and training tutorials AI-readable. Security, Deployment, and Device Support Training for your AI assistant.

Need help with Apple device deployment, security architecture, or support training? AI assistants can help!

Apple Support guides and training tutorials contain comprehensive information about Platform Security, Deployment, and Device Support, but they're difficult for AI assistants to parse and understand. This service makes them accessible in clean, AI-readable Markdown format.

Access 257+ deployment topics, comprehensive security documentation, and 90+ official training tutorials (26hr total: Device Support 14hr 15min + Deployment 11hr 45min) through a simple API or MCP integration.

MCP Integration

Connect your MCP client to access Apple Support guides. Use http://localhost:51345/mcp for local development or your deployed URL for production.

Available Resources

  • support://{guide}/{path}
    Apple Support guide pages in Markdown format
    Example: support://security/secure-enclave-sec59b0b31ff returns Secure Enclave documentation
    Example: support://deployment/intro-to-declarative-device-management-depb1bab77f8 returns Declarative Device Management guide

Available Tools

Support Guide Tools

  • searchAppleSupportGuide
    Search for topics in Apple Platform Security or Deployment guides
    Parameters:
    • guide (enum): "security" or "deployment"
    • query (string): Search query

    Returns list of matching topics with slugs and URLs
  • fetchAppleSupportGuide
    Fetches Apple Support guide pages by guide name and path
    Parameters:
    • guide (enum): "security" or "deployment"
    • path (string): Page path/slug (e.g., "secure-enclave-sec59b0b31ff", "intro-to-declarative-device-management-depb1bab77f8")

    Returns content as Markdown

Device Support Training Tools

  • searchAppleSupportTraining
    Search Device Support training tutorials
    Parameters:
    • query (string): Search query (e.g., "backup", "wifi troubleshooting")
    • platform (enum, optional): "iphone", "ipad", "mac", or "all"

    Searches 40+ tutorials (14hr 15min) covering iPhone, iPad, and Mac support
  • fetchAppleSupportTraining
    Fetch a specific Device Support tutorial by ID
    Parameters:
    • tutorialId (string): Tutorial ID (e.g., "sup005", "sup110")

    Returns full tutorial content as markdown
  • listAppleSupportTrainingCatalog
    Get the Device Support course structure
    Parameters: None
    Returns all volumes, chapters, and 40+ tutorials

Deployment Training Tools

  • searchAppleDeploymentTraining
    Search Deployment & Management training tutorials
    Parameters:
    • query (string): Search query (e.g., "mdm", "configuration profiles")

    Searches 50+ tutorials (11hr 45min) covering MDM and deployment
  • fetchAppleDeploymentTraining
    Fetch a specific Deployment tutorial by ID
    Parameters:
    • tutorialId (string): Tutorial ID (e.g., "dm005", "dm110")

    Returns full tutorial content as markdown
  • listAppleDeploymentTrainingCatalog
    Get the Deployment Training course structure
    Parameters: None
    Returns all volumes, chapters, and 50+ tutorials

Troubleshooting

If you're experiencing connection timeouts or network issues with the MCP server, you may need to configure a proxy. This is particularly common in corporate environments or regions with restricted internet access.

Configure your MCP client to use a proxy by adding environment variables: 

{
  "mcpServers": {
    "supportify": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-remote",
        "http://localhost:51345/mcp"
      ],
      "env": {
        "HTTP_PROXY": "http://proxy.example.com:8080",
        "HTTPS_PROXY": "http://proxy.example.com:8080"
      }
    }  }
}

Replace proxy.example.com:8080 with your actual proxy server details. For authenticated proxies, use the format: http://username:password@proxy.example.com:8080
